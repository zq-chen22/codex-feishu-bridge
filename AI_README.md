1. `README.md` 是冻结的展示页。除版权所有者明确批准的法律或产品定位调整外，不添加补丁说明、实现细节、兼容提示或临时文案。

3. 不记录凭据、个人信息、组织名称、真实聊天 ID、主机路径或私有部署数据。

### 隐私清理

- 删除硬编码的私有 Codex thread 及部署专属标题。
- 移除组织、地点、真实用户名和个人主机路径。
- 飞书群描述改用不可逆绑定标记，不再显示绝对工作路径或原始 thread ID，同时保留旧私有部署的一次性迁移识别能力。
- 日志对平台 ID、UUID、用户主目录、邮箱和常见凭据格式脱敏。
- 数据库默认保留已完成消息正文 30 天并每日清理；已发送 outbox 正文立即清空。
- 宣传图需要匿名化真实头像/昵称/主机信息，并移除来源不明的卡牌素材；旧图不进入公开发布。

### 安全修复

- 默认 Codex 权限由 `never + danger-full-access` 收紧为 `on-request + workspace-write`。
- 默认工作区仅允许飞行桥管理的 workspace，不再默认放行整个用户主目录。
- 飞书端远程 Full Access 默认隐藏并拒绝，只有本机显式配置后才可启用。
- 配置值、Secret 环境变量名、安全目录和文件权限增加检查。
- 详细日志只提升飞行桥自身且已脱敏的 logger；HTTP、WebSocket 和飞书 SDK 保持 `WARNING`，避免调试模式输出请求内部信息。
- 图片代理缓存加入源文件 SHA-256，消除相同大小和时间戳下复用旧代理的完整性风险。
- 继续保留所有者配对、租户校验、消息幂等、文件隔离、软/硬链接拒绝、大小限制和凭据特征扫描。
