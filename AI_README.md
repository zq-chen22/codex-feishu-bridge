1. `README.md` 是冻结的展示页。除版权所有者明确批准的法律或产品定位调整外，不添加补丁说明、实现细节、兼容提示或临时文案。
2. 每次代码、配置、依赖、素材、文档、测试、部署或发布变化，都要在本文件顶部按日期追加记录。
3. 不记录凭据、个人信息、组织名称、真实聊天 ID、主机路径或私有部署数据。
4. 修改 README 后必须生成页面截图供版权所有者确认。
5. 默认安全边界只能收紧；若需要放宽，必须说明威胁、显式开关、迁移方式和测试证据。
