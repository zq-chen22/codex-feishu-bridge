# 飞行桥

让 Codex 留在设备上持续工作，你只需要打开飞书。

</div>

## 我们的宗旨

越接近通用的 AI，越应该用最自然的方式协作。<br>
协调 Codex 项目，最符合人类认知规律的方式，就是**聊天软件**。<br>
**飞书**，是其中实际体验最好的国内方案。我们致力于推动它。

## 我们的功能

<p>
<strong>01 / 不同设备</strong>&ensp;在飞书里协调不同设备上的 Codex<br>
<strong>02 / 状态同步</strong>&ensp;飞书任务中的思考状态、进展与计划，可通过飞书卡片及时同步<br>
<strong>03 / 文件流转</strong>&ensp;支持上传文件、图片和视频；任务完成后，Codex 会发来必要的图片、文件和视频<br>
<strong>04 / 无需梯子</strong>&ensp;手机端无需梯子，也能通过飞书使用 Codex
</p>

## 我们的承诺

<p>
<strong>01 / 放心使用</strong>&ensp;代码公开可审计，安全问题支持私下披露<br>
<strong>02 / 真实稳定</strong>&ensp;实际使用中支持每人每天 100+ 轮次对话<br>
<strong>03 / 无限 DIY</strong>&ensp;功能可交给 Agent 快速扩展，保留充分 DIY 空间
</p>

## 使用方法

<p><strong>10min · 无门槛</strong>&ensp;使用 Codex-5.6 或更高水平的 AI Agent 模型，拉取当前项目并遵循指令即可。</p>
